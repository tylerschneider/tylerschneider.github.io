VRT Museum

Quick Summary

Solo Project • Concepting • Environment • Programming • VR • Modeling • Texturing/Materials • Lighting • Rendering

VRT Museum is a small VR game created entirely by me over about a month. My focus for this project was on creating an appealing environment and lighting in Unity. I also wanted to create fun VR interactivity. The goal of the game is to interact with objects in a sandbox environment to create unique art sculptures and display them.

Features

  • Solo Project - This project was created by me from scratch in Unity
  • Programming - Using C# and XR Interaction Toolkit (preview version)
  • Environment - Created new Models, Textures, and Lighting
  • Interactivity - Raygun object allows interactivity with items that alter your sculpture. Items can interact with machines to change their properties
  • Aesthetic - Combinds Roman architecture and Sci-fi machinery for a contrast between new and old but with a unified, clean look
  • Rendering - Created
