Board Game

Quick Summary

Concepting • Pixel Art • UI • Programming • Animation

Board Game was created as a group game, and my second major project in the GIMM Program. The game was created through Adobe Animate using Actionscript 3. My parts of the project were creating the board, dice rolling, and crane game, including both the visuals and programming.

Contributions

  • Sprite Art - Visuals are sprites created in Photoshop
  • UI - Board UI design and implimentation
  • Custom Boards - Boards are created by the player by placing spaces which are randomized
  • Dice Rolling - 3D Dice roller created entirely with Actionscript
  • Crane Game - 1v1 crane game where the one with the most balls collected wins, also created through Actionscript
