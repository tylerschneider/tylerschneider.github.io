Egyption Conniption

Quick Summary

Environment • Texturing • Modeling • Lighting • Level Design

Egyptian Conniption is a 3D platformer minigame created in Unreal Engine. The project was created over two weeks within a group of four people. My focus for this project was as the Art/Environment lead.

Contributions

  • Art Lead - Handled visual aspects of the project
  • 3D Models - Created all 3D models (except default Unity character) using Blender
  • Textures - Used Substance Designer and Photoshop to create Environment textures
  • Lighting - Learned how to manage lighting and effect systems in Unreal Engine
  • Level Design - Built the level, puzzles and obstacles for the player to make their way through (Object interactivity/movement was mainly created by another group member)
  • Minor Blueprint Contributions - Created and altered some Blueprints using the Unreal Engine Blueprint system
