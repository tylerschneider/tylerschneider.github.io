Spooky Hoops

Quick Summary

Team Lead • Concepting • Lighting • Visual Effects • Music & Sound

Spooky Hoops was created with the goal of implementing machine learning agents to a minigame within Unity. This project was made with a group of four over the course of two weeks. The game's objective is to fly through more hoops than the AI opponent. My focus for this project was as team lead and keeping a consistent halloween theme within every aspect of the game.

Contributions

  • Team Lead - Managed other group members including Artist, Main programmer, and Second programmer/tester
  • Git Management - Created Github page and oversaw merges and changes
  • Lighting - Used Unity lighting & post processing systems
  • Shaders - Created shaders using Universal Render Pipeline shader graph for the Particles and Skybox
  • Particle Systems - Created ghost particles and 3D Candy explosions with Unity particle system
  • Environment layout - Placed and organized objects provided by artists
  • Music & Sound Effects - Created custom music and sound effects for the game using FL Studio and Audacity
  • Difficulty slider - Added a slider to the main menu to control length/difficulty of the game by increasing the hoop requirement
