Pocket Healer

Quick Summary

Solo Project • Concepting • Pixel Art • Programming • Animation

Pocket Healer was created as my first larger game project. The game was created through Adobe Animate using Actionscript 3. This was a solo project and created in around 10 weeks, with all art, programming, and concepts created by me. The game's objective is to play as a healer in a simulated RPG, where you have to use abilites to keep your team alive until they defeat the bosses.

Features

  • Sprite Art - Visuals are all custom sprites created in Photoshop
  • Programming - Game entirely programmed using Actionscript 3
  • Cutscenes - Created cutscenes using Animate's timeline and custom spritework
  • Levels - Multiple different levels with increasing difficutly, including a special final boss with different gameplay
  • Abilities - Multiple abilities with different traits to vary gameplay
